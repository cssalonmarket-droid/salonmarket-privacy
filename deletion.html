⚠️ 보관 데이터의 처리 원칙

법령에 따라 보관되는 데이터는 회원 탈퇴 후 식별 정보를 마스킹 처리한 상태로 보관되며, 보관 기간 만료 시 자동 폐기됩니다. 보관 기간 중 회사는 본인 확인 후 거래 분쟁 해결, 세무 신고 등의 목적 외에는 해당 데이터에 접근하지 않습니다.

3. 부분 데이터 삭제 (계정 유지 + 일부 데이터만 삭제)

계정을 삭제하지 않고 특정 데이터만 삭제하길 원하시는 경우, 다음 방법으로 요청 가능합니다.

  • 업로드 사진 삭제: 마이페이지 → 내 게시물 → 개별 삭제
  • 채팅 내역 삭제: 채팅 화면 → 메뉴 → 채팅방 나가기 (개별 삭제)
  • 위치 정보 수집 중단: 휴대폰 설정 → 살롱마켓 권한 → 위치 OFF
  • 마케팅 알림 수신 중단: 마이페이지 → 알림 설정 → 마케팅 OFF
  • 기타 부분 삭제 요청: cs.salonmarket@gmail.com 으로 이메일 요청

5. 자주 묻는 질문

Q1. 회원 탈퇴 후 같은 이메일로 재가입할 수 있나요?

네, 탈퇴 후 언제든지 재가입 가능합니다. 단, 이전 계정의 활동 기록 / 살롱컬러 등급 등은 복원되지 않으며, 신규 회원으로 시작합니다.

Q2. 거래 진행 중에 탈퇴할 수 있나요?

진행 중인 거래(낙찰, 견적 매칭 등)가 있는 경우 거래 완료 또는 정산 완료 후 탈퇴 가능합니다. 거래 분쟁 발생 시에는 분쟁 해결 후 탈퇴 처리됩니다.

Q3. 탈퇴 처리가 완료됐는지 확인하려면?

처리 완료 시 본인 이메일로 회신을 보내드립니다. 또한 탈퇴 후에는 동일 계정으로 로그인이 불가능하므로 직접 확인 가능합니다.

Q4. 안전결제(에스크로) 도입 시점에 탈퇴하면 어떻게 되나요?

안전결제 시스템 도입 후, 결제대금이 보관 중인 거래가 있는 경우 정산 완료까지 탈퇴가 보류될 수 있습니다. 자세한 내용은 회사 이용약관 제6조 및 제14조 참조.
